# Информационно-справочная система службы такси

Курсовая работа по дисциплине "Современные технологии программирования".

## Как запустить программу
1. Установите PostgreSQL и создайте базу `taxi_db`
2. В файле `src/main/resources/hibernate.cfg.xml` укажите ваши логин и пароль от БД
3. Запустите `JettyLauncher.main()`

## Основные функции
- Управление автомобилями таксопарка
- Учет водителей и их документов
- Обработка заказов такси
- Контроль медицинских и технических осмотров
